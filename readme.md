# VLXTools 工具箱

## 项目介绍
VLXTools 是一款基于纯静态 HTML 开发的多功能工具箱，无需后端服务支持，仅需浏览器即可运行。它集成了多种实用工具，旨在为用户提供便捷、高效的在线工具使用体验，满足日常工作与学习中的各类小需求。

## 功能特点
- **纯静态架构**：无需服务器部署，但是需要浏览在线环境，否则无法加载json内容以渲染菜单。
- **多工具集成**：涵盖文本处理、格式转换、计算工具、编码解码等多种实用功能（可根据实际工具列举，如：JSON 格式化、时间戳转换、URL 编码解码等）。
- **轻量高效**：体积小，加载速度快，无多余广告与冗余内容。
- **离线可用**：一旦加载完成，无需网络连接也能正常使用所有功能。

## 使用方法
1. 克隆或下载本项目到本地；
2. 找到项目根目录下的 `index.html` 文件；
3. 用任意现代浏览器（如 Chrome、Firefox、Edge 等）打开该文件；
4. 在页面中选择所需工具，按照提示进行操作即可。

## 工具列表（部分示例）
- 文本工具：文本去重、大小写转换、字数统计 
- 格式转换：JSON 与 XML 互转、Markdown 预览
- 编码工具：Base64 编码/解码、MD5 加密
- 计算工具：简易计算器、进制转换
- 其他工具：随机密码生成、UUID 生成

## 本地开发与修改
若需自定义工具或修改界面：
1. 直接编辑 `index.html`、`css/style.css`、`js/script.js` 等文件；
2. 新增工具可在 `js` 目录下创建对应功能脚本，并在 `index.html` 中引入；
3. 修改完成后，刷新浏览器即可查看效果。
